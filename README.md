# iUpi – Growth Documentation

##  Descripción
Este repositorio contiene la documentación del proyecto Growth realizado para **iUpi Ahorro & Inversiones**, en el marco de una Simulación Laboral.

El objetivo principal es mejorar la conversión de visitantes a registros de email, trabajando sobre claridad del mensaje, generación de confianza y experimentación por canales digitales.

---

##  Objetivo del Proyecto
Incrementar la tasa de conversión de la landing page, logrando que más usuarios:
- Comprendan rápidamente el valor de iUpi
- Confíen en la marca
- Dejen su email para acceder a la prueba gratuita
